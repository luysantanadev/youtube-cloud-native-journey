Como sobrescrever ao rodar o container

Use a flag -e no docker run para passar variáveis individualmente:

docker run --rm -p 3000:3000 -e APP_ENV=production -e APP_MESSAGE="Viva o K8s!" workshop-envs:0.1.0

Ou use um arquivo .env:

docker run --rm -p 3000:3000 --env-file .env workshop-envs:0.1.0

No Kubernetes, variáveis de ambiente vêm de ConfigMap e Secret, sem alterar a imagem.
